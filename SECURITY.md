## Secret Handling

- Never commit `.env`, API keys, credentials, tokens or private certificates.
- Use `.env.example` for non-sensitive configuration examples.
- Rotate any credential that may have been exposed.
- Treat AI tool prompts and generated context packs as potentially sensitive if they include client or project details.

## Reporting

- Report suspected vulnerabilities to the repository owner or designated security reviewer.
- Do not disclose security issues publicly before review.
