# ShipSeal dokumentációs csomag

Ez a ZIP a korábbi AgentReady projektindító dokumentáció ShipSeal névre átírt és kiegészített változata.

## Tartalom

1. `00_SHIPSEAL_PROJECT_CONTEXT_PROMPT.md` - új projektkontextus és fő prompt
2. `01_CEG_ES_TERMEKLEIRAS_SHIPSEAL.md` - cég- és termékleírás
3. `02_UZLETI_ES_MEGVALOSITASI_TERV.md` - üzleti és megvalósítási terv
4. `03_FORRASLELTAR_ZIP_ELEMZES.md` - forrásleltár a meglévő ZIP-ek alapján
5. `04_FEJLESZTESI_IGENYEK_ES_ROADMAP.md` - fejlesztési igények és roadmap
6. `05_EU_AI_ACT_JOGI_WORKSTREAM.md` - EU AI Act és jogi workstream
7. `06_SHIPSEAL_MVP_OUTPUT_SPEC.md` - MVP output specifikáció
8. `07_FEJLESZTESI_KEZDOPONT_ES_ELSO_SPRINTEK.md` - mivel kezdjük a fejlesztést
9. `ShipSeal_Projektindito_Dokumentacio_OSSZES.md` - összesített Markdown
10. `ShipSeal_Projektindito_Dokumentacio.docx` - szerkeszthető Word-változat

## Használat

A régi `agentready_project_documents.zip` helyett ezt a csomagot érdemes használni projektindító dokumentációként.

A forráskódban a rebrandet külön kell elvégezni: a jelenlegi repo neve és több komponensnév még `agentready` néven szerepel. Az első fejlesztési branch javasolt neve: `shipseal-mvp-delivery-pack`.
