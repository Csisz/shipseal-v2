# Visual Language (75%)

## Motto
Turn software into knowledge.

## Visual Rules
- One primary message per screen.
- Never overwhelm.
- Use motion to explain.
- Use whitespace generously.
- Visual hierarchy before data density.

## Motion
Every animation must represent a real repository event.
No decorative motion.

## Colors
Calm dark canvas.
Bright intelligence accents.
Evidence highlighted consistently.

## Anti-patterns
- KPI walls
- Long reports
- Static dashboards
- Excessive tables
